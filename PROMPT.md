# コーダー依頼用プロンプト

以下の指示に従い、耳つぼダイエットサロン「Anandah」のLPを実装してください。

## 1. 前提資料
プロジェクトルートにある以下の2つのファイルを熟読してください。
- `REQUIREMENTS.md`: 要件定義書（ビジネスゴール、ターゲット）
- `SPEC.md`: 実装指示書（ディレクトリ構成、デザイン、詳細な文言）

## 2. 技術スタック
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## 3. 実装手順
1. **プロジェクトセットアップ**:
   - 現在のディレクトリでNext.jsプロジェクトを展開（または既存をクリーンアップ）
   - `tailwind.config.ts` に `SPEC.md` で定義されたカラーパレットを設定
   - `layout.tsx` に `Noto Sans JP` フォントを適用

2. **コンポーネント実装**:
   - `components/sections/` 配下に各セクションを作成
   - `Header`, `Hero`, `Problem`, `Solution`, `Features`, `Menu`, `Owner`, `Access`, `Footer`
   - スマホユーザー向けに `FixedCTA` (画面下部固定の予約ボタン) を実装

3. **ページ組み立て**:
   - `app/page.tsx` で各セクションを配置

## 4. 注意点
- **モバイルファースト**: スマホでの表示を最優先してください。ターゲット層（40-60代）のため、文字サイズは小さすぎないように配慮すること。
- **画像**: 画像ファイルがない場合は、適切なサイズのプレースホルダー（色付きdivなど）で枠を確保し、後で差し替え可能な状態にしてください。
- **文言**: `SPEC.md` に記載された原稿をそのまま使用してください。

## 5. 成果物
- エラーなくビルド・起動できるソースコード一式
