## 4. 運用・デプロイ手順（重要）

**本プロジェクトは「Vercel手動デプロイ」で運用する。**
GitHub連携による自動更新は使用しない（アカウント制限のため）。

### 更新フロー
1. AIがコードを修正する。
2. 修正後、AIが以下のコマンド実行を依頼する。
3. ユーザーがターミナルで実行して本番反映する。

```bash
npx vercel --prod
```

### 注意点
- **自動更新はされない**: GitHubにPushしても本番には反映されない。必ず上記コマンドが必要。
- **独自ドメイン**: Vercel上で設定可能。URLが変わっても影響なし。
