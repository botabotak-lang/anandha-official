# 実装指示書 (SPEC.md)

## 1. 技術スタック・環境
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Font**: Noto Sans JP (Google Fonts)
- **Icons**: Lucide React

## 2. デザインシステム

### 配色 (tailwind.config.ts)
- **Main Background**: `#FAFAF9` (ウォームホワイト - 生成り色、自然な温かみ)
- **Content Background**: `#FFFFFF` (白 - 清潔感)
- **Primary Text**: `#434343` (墨色 - 真っ黒ではない、柔らかな読みやすさ)
- **Brand Main**: `#5F7161` (セージグリーン - ロゴ由来の癒しの緑)
- **Accent (Gold)**: `#C5A059` (高級感、見出しのアクセント)
- **Accent (Terracotta)**: `#D48E7E` (健康、女性らしさ、ポイント強調)
- **CTA (Button)**: `#D48E7E` (テラコッタ) または `#5F7161` (グリーン)

### フォント設定 (layout.tsx)
- `next/font/google` より `Noto Sans JP` をロード (Weight: 400, 500, 700)
- body全体に適用
- 行間(`leading-relaxed`)を広めに取り、ゆったりとした印象を与える

## 3. ディレクトリ構成
```
app/
  layout.tsx      # 全体レイアウト
  page.tsx        # LPメインページ
components/
  ui/             # Button, Card等の汎用パーツ
  sections/       # 各セクション
    Hero.tsx      # (1) キャッチコピー
    Problem.tsx   # (2) 共感・問題提起
    Solution.tsx  # (3) 解決策・提案
    Features.tsx  # (4) 3つの特徴
    Flow.tsx      # (5) 施術の流れ
    Menu.tsx      # (6) メニュー・料金
    Campaign.tsx  # (7) 初回限定オファー
    Owner.tsx     # (8) プロフィール
    CTA.tsx       # (9) クロージング & 固定フッター
    Footer.tsx
```

## 4. コンテンツ原稿・セクション詳細

### (1) Hero (`components/sections/Hero.tsx`)
- **Background**: 清潔感のあるサロンイメージ、またはリラックスした女性の笑顔
- **Main Copy**:
  > 運動も、我慢も、もう手放しましょう。\
  > 人生最後のダイエットで、心と体を「健康」へ。
- **Sub Copy**:
  > 40代からの身体が変わる。\
  > 無理なく食べて、美しく痩せる\
  > 大人のための耳つぼダイエットサロン Anandha

### (2) Problem (`components/sections/Problem.tsx`)
- **Heading**: 「昔と同じ食事なのに、なぜか太ってしまう」その原因は、年齢による「代謝の低下」かもしれません。
- **Content**:
  > 子育てもひと段落し、これからは自分のための人生を楽しみたい。\
  > そう思っているのに、鏡を見るたびにため息が出てしまうことはありませんか？
  - **List**:
    - 「食事の量は変えていないのに、体重が増え続ける」
    - 「膝や腰が痛くて、運動なんてとてもできない」
    - 「健康診断の数値が悪化して、将来が不安…」
  > いろいろなダイエットを試してはリバウンドを繰り返し、「私の意志が弱いから」とご自身を責めてしまっているかもしれません。\
  > \
  > でも、どうか安心してください。\
  > 痩せにくいのは、あなたの努力が足りないからではありません。\
  > 年齢とともに**「基礎代謝」が落ちてしまい、若い頃のようにエネルギーを消費できなくなっている**のが大きな原因なのです。

### (3) Solution (`components/sections/Solution.tsx`)
- **Heading**: 必要なのは「我慢」ではありません。耳つぼで「代謝」と「食欲」のスイッチを入れましょう。
- **Content**:
  > Anandha（アナンダ）が提供するのは、医学的根拠に基づいた「耳つぼダイエット」です。\
  > \
  > 耳には、自律神経や内臓の働きに関わる重要なツボが集中しています。\
  > ここに小さな「金粒」を貼ることで、以下の2つの効果を促します。
  - **Point 1**: 食欲を自然にコントロール（食欲中枢を穏やかに刺激し、「食べたい」という衝動を落ち着かせます）
  - **Point 2**: 代謝機能を活性化（代謝アップのツボも刺激することで、落ちてしまった「燃やす力」を取り戻します）
  > 針は一切使いません。痛みもなく、目立たない小さな粒を貼るだけ。\
  > また、身体への負担を考え、無理な運動もお勧めしていません。\
  > \
  > 体の内側から「燃えやすい体」へと体質改善を促す。\
  > それが、私たちが提案する「人生最後のダイエット」です。

### (4) Features (`components/sections/Features.tsx`)
- **Heading**: Anandhaが選ばれる3つの理由
- **Cards**:
  1. **つらい運動は必要ありません**
     > ダイエットのために、苦手な運動を無理に始める必要はありません。代謝が落ちている状態で激しく動くと、かえって食欲が増したり、関節を痛める原因になることも。家事や日常の動きだけで十分です。
  2. **無理な食事制限なし＆プロの栄養サポート**
     > 「食べない」ダイエットは筋肉を落とし、リバウンドの元です。Anandhaでは極端な食事制限は行いません。プロのアドバイスのもと、必要な栄養をしっかり摂って「燃やす力」を育てていきます。お酒や外食も、コツさえ掴めば楽しんでいただけます。
  3. **マンツーマンで徹底サポート**
     > 経験豊富なプロのカウンセラーが、あなたと二人三脚で歩みます。体重の減り方や生活習慣は人それぞれ。あなたに合わせたアドバイスで、停滞期も孤独にさせません。

### (5) Flow (`components/sections/Flow.tsx`)
- **Heading**: 卒業まで、無理なく続けられる3ステップ
- **Steps**:
  1. **STEP.01 カウンセリング**
     > 90分ほどで、あなたに最適なダイエットプランをご提案します。期間や料金も事前に分かるのでご安心ください。
  2. **STEP.02 ダイエット開始**
     > 「これならできそう！」と思えたら、お申し込みください。耳つぼ施術は1回15分ほど、週2回お越しください。
  3. **STEP.03 卒業**
     > 目標体重に達したら、卒業です。無料アフターフォローがあるので、リバウンドの心配もありません。

### (6) Menu (`components/sections/Menu.tsx`)
- **Heading**: 明確な料金システムで、安心して通っていただけます。
- **Layout**: 見やすいリスト形式またはカード形式
  - **■カウンセリング（初回無料）**
    - 現在の健康状態・食生活のヒアリング
    - 耳つぼ痩身プログラムの提案・詳細説明

  - **■耳つぼダイエット**
    - 週2回ほどの通院で、身体のバランスを整えていきます。
    - 施術内容：体重測定・食生活チェック、カウンセリング
    - 耳つぼ施術：1回 3,080円（施術2,200円 材料880円）
    - 補助食品：6,264円
