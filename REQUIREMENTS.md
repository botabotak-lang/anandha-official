# 要件定義書 (REQUIREMENTS.md)

## 1. プロジェクト概要
- **プロジェクト名**: Anandha HP制作
- **目的**: チラシを見たユーザーに対する詳細情報の提供と信頼性の担保
- **ターゲット**: 40代後半〜60代女性
    - 子育てがひと段落し、自分に時間とお金をかけられる
    - 「運動したくない」「お酒を楽しみたい」「代謝が落ちた」などの悩みを持つ
    - 健康診断の結果が気になり始めている

## 2. ビジネス要件
- **KGI (最終目標)**: 来店数増加
- **KPI (中間目標)**: 
    - サイト滞在時間の最大化（しっかり読んでもらう）
    - LINE公式アカウント登録数
    - 耳つぼ体験予約数
    - **SEO目標**: 「藤枝市 耳つぼ」で検索順位1位を獲得する
- **提供価値**: 
    - 「痩せない」悩みへの共感
    - 「運動なし・食事制限なし」という安心感の提供
    - オーナーの人柄による信頼感

## 3. 機能要件
- **ページ構成**: シングルページ（LP形式）
- **必須セクション**:
    1. **Hero (FV)**: キャッチコピー、イメージ画像
    2. **Concept**: 悩みへの共感、解決策（耳つぼ）の提示
    3. **Features**: Anandhaの強み（運動なし、3食OKなど）
    4. **Menu/Price**: 料金表（体験価格の強調）
    5. **Owner**: プロフィール、顔写真、想い
    6. **Access**: 店舗情報、地図（Google Maps）
    7. **CTA**: LINE登録ボタン、電話予約ボタン
- **お問い合わせフォーム**: なし（LINE/電話へ誘導）

## 4. 非機能要件
- **デザイン**:
    - **テイスト**: 落ち着き、高級感、清潔感、信頼感
    - **文字サイズ**: ターゲット層に合わせて大きめに設定
    - **配色**: 
        - ベース: ウォームホワイト (#FAFAF9) - 和紙のような生成り色、自然な風合い
        - メイン: セージグリーン (#5F7161) - ロゴに合わせた植物の緑、癒し
        - アクセント: ゴールド (#C5A059) - 高級感 / テラコッタ (#D48E7E) - 健康的な血色感
- **パフォーマンス**: ストレスのない表示速度
- **SEO対策**: 
    - **地域キーワード**: 「藤枝市」「焼津市」「島田市」などを適切に盛り込む
    - **構造化データ**: LocalBusinessスキーマを実装し、Googleマップ（MEO）との連携を強化
    - **メタデータ**: title, descriptionに「藤枝市」「耳つぼダイエット」を必ず含める
- **レスポンシブ**: スマホ表示を最優先（モバイルファースト）

## 5. 運用・インフラ要件
- **開発言語**: Next.js (App Router) + Tailwind CSS
- **ホスティング**: Vercel (初期ドメイン利用)
- **更新フロー**: 静的LPとして運用。更新時は開発者に依頼。
